# Data in Motion – Cross Platform Audience Growth Analysis

## Overview
This project analyzes LinkedIn and YouTube performance data for Data in Motion to identify the key drivers of audience growth, engagement, and subscriber conversion.

The analysis combines SQL, Python, and Tableau to transform raw social media data into actionable insights that support content strategy decisions.

## Tools Used
Python (Pandas, Matplotlib, Seaborn)  
SQL  
Tableau  
Excel

## Key Insights
• LinkedIn engagement rate reached 1.45%, exceeding typical B2B benchmarks  
• Niche content produced 65% higher engagement than broad reach posts  
• Conversion increased from 0.10% to 6.1%  
• Subscriber growth is strongly correlated with impressions and watch time

## Project Structure
